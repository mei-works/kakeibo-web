# 家計簿Webアプリ

## 概要

Flask・HTML/CSS・JavaScriptを用いて開発した家計簿Webアプリです。

**公開URL**
https://kakeibo-web-mk7z.onrender.com

## 使用技術

* Python
* Flask
* HTML/CSS
* JavaScript
* Chart.js
* Matplotlib
* GitHub
* Render

## 主な機能

* CSVファイルの読み込み
* 月別の収入・支出集計
* 収支バランスの計算
* カテゴリ別支出の円グラフ表示
* 月別収入・支出の棒グラフ表示
* Webブラウザ上での閲覧

## 工夫した点

* Pythonで家計簿データを集計
* Chart.jsを用いたデータの可視化
* Renderを利用したデプロイ・公開
